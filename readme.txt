Door -> ни претставува вратата т.е. таа праќа логови за се што ќе се случи при нејзино отварање (кој ја отворил, која просторија итн.)
ProfessorLog -> ни претставува за чување на логови само за професори кои што пристапуваат кон простории и се чуваат во Professor_log.csv документот
StudentLog -> ни дава логови само доколку студент пристапи до канцеларија
StudentLogCSV -> исто како за ProfessorLog само што наместо за професор, чува логови во Student_log.csv
Unlocker -> овде ни е симулирано пристапот до вратата, т.е. се избира дали е професор или студент, внесува име и кој број на просторија
